# 🎮 Tic Tac Toe Game Using Python Tkinter

A modern and interactive **Tic Tac Toe Game GUI Application** developed using **Python and Tkinter**. This project was created during my **internship period** as a practical implementation task to improve my skills in Python programming, GUI development, and game logic implementation.

The application provides a user-friendly interface where two players can play Tic Tac Toe in a clean dark-themed graphical environment.

## 📌 Project Overview

This Tic Tac Toe game is a two-player desktop application where players take turns placing **X** and **O** on a 3×3 grid. The program automatically detects the winner or declares a draw when all positions are filled.

This project helped me understand:

- GUI application development using Tkinter
- Event-driven programming
- Button handling and user interaction
- Game logic implementation
- Managing application states using Python

---

## ✨ Features

- Interactive 3×3 Tic Tac Toe board
- Two-player gameplay (X vs O)
- Automatic winner detection
- Draw match detection
- Restart game functionality
- Modern dark-themed user interface
- Different colors for X and O players
- Simple and lightweight application
